# Глоссарий: внутренние термины → простой язык / Glossary: internal terms → plain language

Правило сайта: на читательских страницах (главная, цепь аргумента, спорные вопросы, карта,
таблица утверждений) используются ТОЛЬКО простые эквиваленты из правой колонки. Каждый
эквивалент можно один раз ввести с пояснением в одну строку. Полная терминология живёт
только на странице «Как проверяли» (справочный блок) и внутри скачиваемого датасета.

Site rule: reader-facing pages use ONLY the plain equivalents. Each may be introduced once
with a one-line explanation. Full terminology lives only on the Method page and inside the
downloadable dataset.

## Объекты исследования / Study objects

| Термин пайплайна | Просто по-русски | Plain English |
|---|---|---|
| claim | утверждение (проверяемое высказывание книги) | a claim (a checkable statement in the book) |
| attributed quote | цитата, приписанная историческому лицу | a quotation the book attributes to a historical figure |
| chapter / paragraph ref | глава и абзац книги, откуда взято утверждение | the chapter and paragraph the claim comes from |
| thesis chain | несущая цепь аргумента книги (7 звеньев) | the book's argument chain (7 links) |
| narrative arc | часть сюжета книги | a section of the book's storyline |
| dossier | спорный вопрос (пачка связанных утверждений, разобранных вместе) | a disputed question (a bundle of related claims examined together) |

## Стадии проверки / Pipeline stages

| Термин пайплайна | Просто по-русски | Plain English |
|---|---|---|
| extraction | выписали из книги все проверяемые утверждения | we copied out every checkable claim from the book |
| verification | сверили каждое с работами историков | we checked each claim against historians' work |
| adversarial defense / defender | отдельная проверка «адвокатом книги»: его задача — оспорить каждое обвинение | a separate "defense attorney" pass: its job is to challenge every accusation |
| adjudication | разбор спорных вопросов по заранее принятым правилам | a review of the disputed questions under rules fixed in advance |
| charter (frozen) | правила разбора, записанные и замороженные до его начала | the review rules, written down and frozen before the review began |
| impact re-score | пересчёт веса ошибок после разбора | re-weighing the errors after the review |
| synthesis | сведение всех результатов в общую картину | pulling all results into one picture |

## Итоги по утверждению (первый круг) / First-round outcomes

| Вердикт пайплайна | Просто по-русски | Plain English |
|---|---|---|
| supported | подтверждено | confirmed |
| contradicted | противоречит данным историков | contradicted by the historical record |
| disputed | наука спорит — вопрос ушёл на отдельный разбор | historians disagree — sent to the review of disputed questions |
| anachronistic | ошибка эпохи: книга повторяет источники XIX века, которые наука с тех пор пересмотрела | an error of its era: the book repeats 19th-century sources that scholarship has since revised |
| misquotation | искажённая цитата | a distorted quotation |
| unverifiable | вне исторической проверки (богословие, сверхъестественное, будущее) | beyond historical checking (theology, the supernatural, the future) |

## Итоги разбора спорного (второй круг) / Disputed-question outcomes

| Решение | Просто по-русски | Plain English |
|---|---|---|
| well-supported | надёжно подтверждено | solidly confirmed |
| probable | скорее верно | more likely true than not |
| genuinely-open | наука всерьёз спорит: сильные доводы с обеих сторон | genuinely open: strong arguments on both sides |
| improbable | скорее неверно | more likely false than true |
| discredited | опровергнуто документами | refuted by documents |
| out-of-scope → conditional premise | держится на посылке, которую история не проверяет | rests on a premise history cannot test |
| ruling as written | оценка формулировки, как она напечатана в книге | how the sentence fares exactly as printed |
| ruling weakened / weakened version | оценка осторожной версии: то же утверждение без «все», «никогда», «единственно» | how a careful version fares: the same claim without "all", "never", "only" |
| framing gap | разрыв между формулировкой и сутью: напечатанный текст не выдержал, осторожная версия подтверждена | the gap between wording and substance: the printed absolute does not hold up, the careful version is confirmed |
| quantifier audit | проверка слов «все», «никогда», «ни один», «единственный» | an audit of the words "all", "never", "not one", "only" |
| minority engagement | разбор сильнейшего довода противоположной стороны | engaging the strongest opposing argument |
| would_change_this | какая находка изменила бы этот вывод | what discovery would change this conclusion |
| confidence | насколько твёрд этот вывод | how firm the conclusion is |
| source_quality | качество источников: научные работы или только справочники | source quality: academic studies or reference works only |
| direction distribution | счёт решений «за» и «против» книги — виден любой перекос | the tally of rulings for and against the book — any drift is visible |

## Классы и веса ошибок / Error classes and weights

| Термин | Просто по-русски | Plain English |
|---|---|---|
| failing component: fact | не выдержал факт (событие, дата, число, цитата) | the fact failed (an event, date, number, or quotation) |
| failing component: frame | не выдержала подача: оценка или обобщение поверх фактов | the framing failed: a judgment or generalization laid over the facts |
| failing component: motive | не выдержал приписанный мотив: книга называет намерение, которого документы не фиксируют | the attributed motive failed: the book names an intention no document records |
| load-bearing | несущее утверждение: на нём держится аргумент главы или книги | a load-bearing claim: the chapter's or book's argument rests on it |
| severity | насколько серьёзна сама ошибка | how serious the error itself is |
| centrality | насколько важное место утверждение занимает в аргументе | how central the claim is to the argument |
| impact score | вес ошибки: серьёзность × важность места | the error's weight: seriousness × centrality |
| chain-inherited | повтор одной ошибки: несколько мест книги стоят на одном и том же корне | a repeat of one error: several passages rest on the same root |
| independent failure point | самостоятельная ошибка (не повтор) | an independent error (not a repeat) |
| period_standard: period-scholarship | так писала наука того времени | the era's own scholarship said the same |
| period_standard: expositor-tradition | так писали толкователи пророчеств — наука того времени уже знала иначе | only the prophecy-expositor tradition said so — the era's scholarship already knew otherwise |
| subtraction analysis | «если убрать ошибочное — что останется» | "remove what failed — what remains" |
| honest boundary | честная граница: что это исследование не решает | the honest boundary: what this study does not decide |

## Замечания по употреблению / Usage notes

- «Вердикт» и «решение» на читательских страницах допустимы как обычные русские слова,
  но без кодов и внутренних имён категорий.
- Идентификаторы утверждений (GC-18-060) на читательских страницах показываются как
  «гл. 18, утв. 60» с раскрытием полного ID в деталях; в таблице и датасете — полный ID.
- Приёмочный тест: каждую страницу можно прочитать вслух человеку, никогда не слышавшему
  о проекте; если фраза требует глоссария — переписать фразу.

## Градация формулировок / Wording gradation

Сила глагола обязана соответствовать силе решения; батальные метафоры запрещены.

| Решение | Допустимая формулировка | Plain English |
|---|---|---|
| discredited / contradicted | «опровергнуто источниками», «опровергнуто документами» | "refuted by the sources/documents" |
| improbable | «скорее неверно», «маловероятно в напечатанном виде» | "more likely false", "improbable as printed" |
| любой обвинительный итог (зонт) | «не подтвердилось», «не выдержало проверки» | "did not hold up" |
| разрушение конструкции | «разрушена» — ТОЛЬКО там, где данные это заслужили (историческая конструкция звена 2: тысячелетний разрыв) | "dismantled", only where the data earned it |

Запрещены: «пало», «рухнул», «выжил», «устоял» и прочая батальная лексика.
Исключение: спокойное «Что выстояло» в заголовках секций допустимо.
Banned: "fell", "collapsed", "survived", battlefield metaphors in general;
the calm "What stood" in section headers is the one allowed exception.
