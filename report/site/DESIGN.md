# Дизайн-система сайта GCHAR

Один экран задачи: серьёзное текстовое исследование книги 1911 года, читатель — обычный
человек. Настроение — редакционный лонгрид (The Pudding / NYT Interactive), не дашборд.
Дизайн строит доверие: сдержанность везде, смелость — в одном месте (цепь аргумента).

## Палитра

Тон подобран от предмета: ледериновые переплёты и бумага научной печати начала XX века.
Единственная светлая тема; фон тонированный, не белый.

| Токен | Hex | Роль |
|---|---|---|
| paper | #F2F1E8 | фон страницы («бумага под холодным светом») |
| paper-deep | #E9E7DA | панели, карточки |
| line | #D8D4C4 | линейки, рамки |
| ink | #262A22 | основной текст (контраст 12.9:1) |
| ink-soft | #565B4E | вторичный текст (6.2:1) |
| binding | #2E4B3F | «переплётная зелень»: ссылки, акценты хрома (8.4:1) |
| brass | #8C7349 | латунные волосяные линии; только декор, не текст |

## Шкала вердиктов (единая для всего сайта)

Divergent-структура: холодная рука «подтверждено», нейтральная середина «спор»,
тёплая рука с нарастающей темнотой «ошибка эпохи → скорее неверно → опровергнуто».
Прогнана через validate_palette.js (dataviz skill): светлотная полоса PASS,
CVD-разделение PASS (все пары, худшая ΔE 13.1 ≥ 12). Два нейтрала (середина и
«вне проверки») намеренно серые; «вне проверки» дополнительно штрихуется.
Цвет никогда не единственный канал: всюду подпись, легенда или тултип.

| Категория | Заливка (графика) | Текст (≥4.5:1) |
|---|---|---|
| подтверждено / well-supported / supported | #005E96 | #00517F |
| скорее верно / probable | #3F90C4 | #1A5E88 |
| наука спорит / genuinely-open / disputed | #6E6A5E | #5C5850 |
| ошибка эпохи / anachronistic | #D9A018 | #7A5A0A |
| скорее неверно / improbable | #AE4A0C | #96400A |
| опровергнуто / discredited / contradicted | #8A3318 | #7C2D15 |
| непроверяемая посылка / conditional premise | #A1568A | #8A4372 |
| вне проверки / unverifiable, out-of-scope | #A9A395 (+штриховка) | #6E6960 |

Заливка охры (2.1:1) и серого-призрака (2.2:1) ниже 3:1 — по правилу dataviz это
законно только с каналом подстраховки: он всегда есть (подписи/легенда/таблица).

Статусы звеньев цепи используют те же токены: цело=supported, повреждено=improbable,
условная посылка=conditional, вне проверки=unverifiable.

## Типографика

| Роль | Шрифт | Почему |
|---|---|---|
| Дисплей (заголовки, крупные числа) | Old Standard TT 400/700 | «Modern»-антиква научной печати рубежа XIX–XX веков — мир самой книги; полная кириллица |
| Текст | IBM Plex Sans 400/500/600 | нейтральный гуманистический гротеск, полная кириллица |
| Данные (ID, подписи, таблицы) | IBM Plex Mono 400/500 | регистр «архивной описи»; полная кириллица |

Самохостинг через @fontsource; кириллические сабсеты проверены в пакетах.

## Подпись дизайна

Цепь тезиса: 7 кованых звеньев ручным SVG. Повреждённые звенья несут видимый разлом
штриха; звено на непроверяемой посылке — пунктирный контур; «вне проверки» — призрачное.
Ховер поднимает звено и показывает комментарий синтеза; клик ведёт на страницу звена.

## Движение

framer-motion везде; scroll-reveal (подъём 8–12px + fade), счётчики чисел,
плавные переходы. Никакого параллакса и декоративного движения.
prefers-reduced-motion отключает всё (значения показываются сразу).

## Правило языка

Читательские страницы — только простые эквиваленты из content/glossary.md.
Терминология пайплайна — только на /method и в датасете.
